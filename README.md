# Customer Revenue Dashboard 
Data analytic project showcasing customer behavior analysis using python,sql and power BI
Customer Revenue Dashboard 

An end-to-end Data Analytics project developed using Python, Jupyter Notebook, PostgreSQL, pgAdmin 4, and Power BI to analyze customer shopping behavior and generate actionable business insights through SQL analysis and interactive dashboards.

Project Overview

This project focuses on customer shopping behavior analysis to identify purchasing trends, customer segments, and revenue patterns using data analytics and business intelligence techniques.

The project demonstrates practical skills in:

Data Cleaning & Preprocessing
SQL Querying
Exploratory Data Analysis (EDA)
Dashboard Development
Business Insight Generation

Tech Stack
Python
Pandas
Jupyter Notebook
PostgreSQL
pgAdmin 4
Power BI


Key Features
-Data Cleaning & Preparation
Processed and cleaned raw customer shopping data
Standardized dataset for analysis
Handled missing values and duplicates
-SQL-Based Analysis

Performed analytical queries to evaluate:

Total Revenue
Average Purchase Amount
Customer Purchase Trends
Repeat Buyers
Subscription-Based Customers
Revenue by Age Group
-Interactive Power BI Dashboard

Developed a dashboard featuring:

KPI Metrics
Revenue Insights
Customer Segmentation
Purchase Analysis
Review Rating Insights
Interactive Filters & Visualizations

Business Insights
Identified high-value customer segments
Analyzed customer purchasing behavior patterns
Evaluated the impact of subscriptions on revenue
Compared spending trends across demographics
Generated actionable business insights through visualization
